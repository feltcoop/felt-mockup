# felt

> free and open community software
> for empowering people and democratic organizations
> :green_heart: tech that feels good

**work in progress** [felt.dev](https://felt.dev)

## develop

```bash
npm i
npm run ts # gogo typescript
npm run tw # open another shell
npm run dev # browse to localhost:3000 or whatever it says
```
